# Milestone Audit: v1.1 Packaging & Adoption

## Verdict

Passed. v1.1 meets its definition of done: installable binary packaging, repo-local agent setup, first-pack adoption, documentation, and release-gate smoke proof are implemented, verified, and wired together.

The first integration check found two release-gate hardening gaps. Both were fixed before this audit was marked passed:

- `scripts/release-gate.sh` no longer defaults `CTXPACK_ALLOW_DIRTY=1`; clean-checkout enforcement remains the default through `scripts/release-package.sh`.
- `scripts/smoke-mcp-protocol.sh` no longer writes a temporary `ctxpack_smoke_current_diff_*` file into the target repository; it uses an isolated temporary git repo for `current_diff` proof.
