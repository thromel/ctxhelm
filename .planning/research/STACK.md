# Stack Research: v2.2 Release & Distribution Hardening

## Current Stack

- Rust workspace and CLI binary.
- Shell scripts for release packaging, release gates, docs checks, and smokes.
- Cargo integration tests for release-script contracts.
- Markdown docs for install, release, troubleshooting, and agent setup.

## Additions

- Source-free release manifest and proof-bundle artifacts.
- Clean extraction verification script or smoke.
- Package metadata templates for future distribution.
- Release candidate status/checklist file format.

## Not Added Yet

- Homebrew tap publishing.
- crates.io publishing.
- Self-update.
- Signed installers or notarization.
- Hosted release service.
