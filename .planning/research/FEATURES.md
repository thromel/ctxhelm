# Feature Research: v2.2 Release & Distribution Hardening

## Table Stakes

- Clean release gate from a clean checkout.
- Release archives and checksums.
- Artifact leakage audit.
- Install and upgrade instructions.
- Troubleshooting flow.
- Public first-pack demo.
- Release readiness checklist.

## Differentiators

- Source-free release proof bundle tying together privacy, smokes, benchmark proof, and client proof status.
- Explicit distinction between deterministic MCP protocol proof and optional real-client proof.
- Release metadata that prepares future package-manager distribution without requiring it now.

## Anti-Features

- Silent global agent configuration mutation.
- Publishing as part of the validation command.
- Claiming signed installers or hosted sync before they exist.
